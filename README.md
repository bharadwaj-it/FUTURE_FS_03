# Forge Fitness Website

## 📌 Project Description

This is a professional website developed for a local gym named Forge Fitness. The goal of this project is to create an online presence for the business and attract more customers.

## 🚀 Features

* Responsive design (mobile-friendly)
* Modern dark UI
* Membership plans
* Services section
* Contact form with popup
* Smooth navigation

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎯 Purpose

Many local gyms do not have an online presence. This website helps showcase services, pricing, and contact details to reach more customers digitally.

## 📢 Pitch

This website helps your gym get more visibility online, attract new customers, and present your services professionally.

## ⚠️ Note
Forge Fitness is a conceptual business created for demonstration purposes.
